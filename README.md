# 💳 Loan Defaulter Prediction

A Machine Learning project that predicts whether a borrower is likely to default on a loan based on financial, demographic, and credit-related factors.

## 📌 Project Overview

Loan default prediction is an important problem in the banking and financial sector. Identifying high-risk borrowers helps financial institutions make informed lending decisions, reduce financial losses, and improve risk management.

This project uses data preprocessing, exploratory data analysis (EDA), and machine learning techniques to predict loan default status.

---

## 🎯 Objectives

* Analyze borrower financial and credit information.
* Identify factors influencing loan defaults.
* Perform data cleaning and exploratory data analysis.
* Build a predictive machine learning model.
* Evaluate model performance using appropriate metrics.

---

## 📊 Dataset Features

| Feature          | Description                                   |
| ---------------- | --------------------------------------------- |
| Age              | Age of the borrower                           |
| Income           | Annual income of the borrower                 |
| Loan_Amount      | Amount of loan taken                          |
| Credit_Score     | Credit score of the borrower                  |
| Employment_Years | Years of employment                           |
| Education_Level  | Educational qualification                     |
| Housing_Status   | Housing ownership status                      |
| Default          | Target variable (0 = No Default, 1 = Default) |
